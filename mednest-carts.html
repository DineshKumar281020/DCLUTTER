AnesthesiaSMART

Patient-Oriented Mobile Medication Management for the Operating Room

MedNest Carts

MedNest Carts is a modular type Smart Drawer Unit that can adapt to the areas of hospitals with different characteristics and provides flexibility and scalability in medication management. The internal chassis is metal reinforced. The number of drawers and drawer sizes can be configured according to user needs.

MedNest Carts is used to close the medication management loop.

System Features

  • Supports Medication Cassette Exchange System.
  • MedNest can be mounted side by side or on top of each other through brackets.
  • Integration with ERP/HIS.
  • Each cart can be equipped with up to two Drawer Chests.
  • Each drawer is configured patient-specific to ensure maximum security.
  • Adaptable to various areas within the hospital.
  • Each module consists of 1-12 drawers and can be configured in 1 size, 2 size, and 4 size single deep or 2 size double deep drawers.

DrawerWizard Patient Verification Module

DrawerWizard is a software solution developed to use together with MedNest Carts.

DrawerWizard functionality provides to assign, unassign, reassign patients and/or items to MedNest Carts drawers; and open patients or item storage drawers of the MedNest Carts unit.

DrawerWizard software enables the patient-based drawers to open/close automatically. Since the drawers can only be opened by the corresponding patient, the application complies with the elimination of medication administration errors.

SMARTTrack Bedside Verification Module

SMARTTrack is a Bedside Verification Module that can manage MedNest Carts patient-based drawers. SMARTTrack improves patient safety and tracks all medication administrations to the patient. It provides confirmation of patient, medication order, and administration of medication. It is the final step of the closed-loop medication management. SMARTTrack has ADT, CPOE, MAR Charting, and Active Directory Integration capabilities.
